# Vendor Medical Trading W.L.L. — Website

React-based corporate website for Vendor Medical Trading W.L.L., Doha, Qatar.

## Tech Stack
- React 18 (Create React App)
- React Router DOM v6
- React Icons
- CSS (no Tailwind — plain CSS with custom properties)
- Google Fonts: Syne + Inter

## Quick Start (Local Development)

```bash
npm install
npm start
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
```

Output goes to `/build` folder. Upload this folder to your host.

## Pages
- `/` — Home
- `/about` — About Us
- `/services` — Services (Medical + Trading Division)
- `/contact` — Contact Us
